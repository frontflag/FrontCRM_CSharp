# 微信扫码登录集成说明

## 概述

本系统实现了**自主绑定模式**的微信扫码登录功能：
- 新用户需先使用账号密码登录，然后在个人中心绑定微信
- 绑定后，下次即可使用微信扫码快速登录
- 未绑定的微信扫码会提示"请先登录绑定"

---

## 部署步骤

### 1. 执行数据库迁移

```bash
# 执行SQL迁移脚本
psql -h localhost -U postgres -d FrontCRM -f update_wechat_auth.sql
```

### 2. 配置微信参数

编辑 `CRM.API/appsettings.json`：

```json
{
  "Wechat": {
    "AppId": "wx_xxxxxxxxxxxxxxxx",      // 微信公众号AppId
    "AppSecret": "xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx",  // 微信公众号AppSecret
    "CallbackDomain": "https://your-domain.com",      // 回调域名
    "QrCodeExpireSeconds": 300          // 二维码过期时间（秒）
  }
}
```

> **注意**：需要在[微信公众平台](https://mp.weixin.qq.com/)注册服务号，并配置回调域名。

### 3. 配置微信回调URL

在微信公众平台后台 → 开发 → 接口权限 → 网页授权：
- 设置 **授权回调页面域名** 为你的域名
- 例如：`your-domain.com`

## API 接口列表

### 扫码登录接口

| 接口 | 方法 | 说明 |
|:---|:---|:---|
| `/api/v1/auth/wechat/qrcode` | POST | 获取登录二维码 |
| `/api/v1/auth/wechat/status/{ticket}` | GET | 查询登录状态（轮询） |
| `/api/v1/auth/wechat/callback` | GET | 微信回调（微信服务器调用） |

### 微信绑定接口（需登录）

| 接口 | 方法 | 说明 |
|:---|:---|:---|
| `/api/v1/auth/wechat/bind-info` | GET | 获取当前绑定信息 |
| `/api/v1/auth/wechat/bind-qrcode` | POST | 生成绑定二维码 |
| `/api/v1/auth/wechat/bind-status/{bindId}` | GET | 查询绑定状态（轮询） |
| `/api/v1/auth/wechat/bind-callback` | GET | 绑定回调（微信服务器调用） |
| `/api/v1/auth/wechat/unbind` | POST | 解除微信绑定 |

## 用户操作流程

### 首次使用

```
1. 访问登录页
2. 选择"账号密码登录"
3. 输入账号密码登录系统
4. 进入"个人设置" → "微信绑定"
5. 点击"生成绑定二维码"
6. 使用微信扫码确认绑定
7. 绑定成功
```

### 后续登录

```
1. 访问登录页
2. 选择"微信扫码登录"
3. 使用微信扫码
4. 登录成功，进入系统
```

## 状态码说明

### 登录状态码

| 状态码 | 含义 | 说明 |
|:---|:---|:---|
| 0 | 待扫码 | 等待用户扫码 |
| 1 | 已扫码 | 用户已扫码，等待确认 |
| 2 | 登录成功 | 认证通过，返回token |
| 3 | 已过期 | 二维码已过期 |
| 4 | 已取消 | 用户取消授权 |
| 5 | 未绑定 | 微信未绑定系统账号 |

### 绑定状态码

| 状态码 | 含义 | 说明 |
|:---|:---|:---|
| pending | 待扫码 | 等待用户扫码 |
| scanned | 已扫码 | 用户已扫码 |
| success | 绑定成功 | 绑定完成 |
| expired | 已过期 | 二维码已过期 |

## 开发注意事项

1. **微信OAuth授权**：绑定和登录都使用微信OAuth授权流程
2. **轮询机制**：前端需要每2秒轮询一次状态
3. **Token复用**：登录成功后复用原有的JWT Token生成逻辑
4. **安全性**：票据和绑定请求都有过期时间，防止重放攻击

## 常见问题

### Q: 微信公众号类型要求？
A: 需要使用**服务号**（订阅号不支持网页授权）。

### Q: 回调域名如何配置？
A: 需要在微信公众平台 → 开发 → 接口权限 → 网页服务 → 网页授权 中设置。

### Q: 一个微信可以绑定多个账号吗？
A: 不可以，一个微信只能绑定一个系统账号。

### Q: 如何解绑微信？
A: 登录系统 → 个人设置 → 微信绑定 → 点击"解除绑定"。
