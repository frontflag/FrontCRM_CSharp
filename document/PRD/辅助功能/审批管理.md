# 审批管理

审批建立独立的数据库表

# 功能要求

- 区分 待审核，审核通过，审核拒绝
- 审核记录 与 业务类型，业务单号的 关联关系
- 提交人，提交日期，提交备注
- 审核人，审核时间，审核意见
- 提交审核时，可上传多个附件
- 默认上级审核下属
- 支持指定的业务类型由特定角色审批。（比如跨部门审核）
- 默认一级审核
- 支持多级审核，可设定某个级别由某个角色审批

# 界面要求

## 提交审核界面

提交审核界面为弹窗，可以输入提交备注，上传多个附件

## 审核界面

**审核界面为单独窗口**

窗口分为2部分

- 第1部分是审核的基本操作内容
- 第2部分是待审核 业务相关信息
    - 不同业务类型，展示的信息内容会不同

**审核统计面板**

审核界面有统计面板，展示 待审核数量，已审核数量，审核拒绝数量
