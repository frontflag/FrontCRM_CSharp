# 文档上传组件规范

本文档沉淀**业务详情页「文档」Tab** 及共用组件中，与**文档上传、待选文件列表、关联文档展示、拖放区域**相关的 UI 与交互约定；实现以当前代码为准，评审与迭代时以此对齐。

---

## 1. 适用范围与参考实现

| 项目 | 说明 |
|------|------|
| 适用场景 | 详情页内「文档 / 附件」区域：上传待提交文件、展示已关联文档、刷新列表 |
| 上传组件 | `CRM.Web/src/components/Document/DocumentUploadPanel.vue` |
| 列表组件 | `CRM.Web/src/components/Document/DocumentListPanel.vue` |
| 整 Tab 拖放（可选） | 业务页将上传区与列表包在同一容器内，容器级 `drop` 转发到上传组件，例如 `CRM.Web/src/views/RFQ/PurchaseOrderDetail.vue` 中 `.doc-tab-content` + `DocumentUploadPanel` 的 `addDroppedFiles` |

## 2. 上传区（DocumentUploadPanel）— 布局与视觉

### 2.1 主工具行（`.upload-toolbar`）

同一行内从左到右依次为：

1. **拖拽 / 点击选择区**（`.upload-area`）
2. **操作按钮**：主按钮「上传」、有已选文件时显示「清空」

约定：

- 主行使用横向 `flex`，**垂直方向居中对齐**（`align-items: center`）。
- 主行最小高度为 `52px`，上传区最小高度同为 `52px`，整体相较旧版约缩小 `1/3`。

### 2.2 备注输入

- 备注输入不在主工具行中展示。
- 仅当 `remarkAllowed === true` 且存在待上传文件（`selectedFiles.length > 0`）时显示。
- 展示位置：**待上传文件列表下方**（`.remark-below`）。
- **不展示**外置字段标题「备注」；语义仅通过控件 **placeholder** 表达。
- **占位文案（固定）**：`请输入备注，选填，最多256字`
- 使用 `maxlength="256"`，并保留字数统计（如 `show-word-limit`）。
- 备注随上传请求一并提交；无内容时可不传或传空（由接口约定）。

### 2.3 拖拽 / 点击选择区（`.upload-area`）

- **不展示**文件夹类装饰图标（如 emoji 图标）。
- 上传组件外层（`.document-upload-panel`）默认显示浅灰色边框：`1px solid #dcdfe6`。
- 宽度为**内容自适应**（`flex: 0 0 auto`），主提示文案单行不换行（`white-space: nowrap`），避免无意义拉宽。
- 主文案与辅文案同一行并排：`拖拽文件到此处，或` + 「点击选择」 + `单次最多 {{ maxFiles }} 个，单文件不超过 {{ maxSizeMb }}MB`。
- 文案字号统一为 `13px`（与页签标题文字一致）。

### 2.4 待上传文件列表（`.file-list`）

- 有已选文件时展示在工具行**下方**。
- **布局**：`flex` + `flex-wrap`，每组为一块「标签式」区域（文件名 + 人类可读大小 + 删除），**多组在同一行横向排列，空间不足时换行**。
- **组间距**：使用 `gap`（行、列）与轻微背景区分组，保证组与组之间间隔明显。
- 文件名过长：`max-width` + 省略号，避免撑破布局。

### 2.5 拖拽高亮

- 在**上传组件根节点**上监听 `dragover` / `dragleave` / `drop`（使用 `preventDefault` + 深度计数避免子节点抖动），进入拖放时增加修饰类（如 `document-upload-panel--dragging`）：边框高亮 + 浅背景，提示当前区域可接收文件。

## 3. 关联文档列表（DocumentListPanel）

### 3.1 工具栏

- 标题文案：**关联文档**。
- **「刷新」按钮**紧挨标题右侧（同一行，`flex-start` + `gap`），**不要**使用 `space-between` 把刷新推到行末最右侧。

### 3.2 展示模式：`view-mode`

| 取值 | 用途 | 展示约定 |
|------|------|----------|
| `list` | 订单类详情等以「信息密度」为主的场景 | 见 §3.3 |
| `grid` | 客户/供应商等需要缩略图卡片感的场景 | 保留缩略图 + 名称 + 元信息 + 操作纵向卡片布局 |

`viewMode` 未传或为非 `grid` 时，按 **list** 逻辑处理（与代码中 `isListView = viewMode !== 'grid'` 一致）。

### 3.3 列表模式（`list`）— 每条文档一块「横向信息条」

每条文档卡片（`.doc-card--list`）内，**主行**从左到右依次为：

1. **文件名**（`originalFileName`，过长省略，`max-width` 约束）
2. **上传时间**（`createTime`，格式化展示）
3. **字节数**：原始 `fileSize` 取整后 **千分位** + 后缀 **`字节`**（例如 `1,234,567 字节`）
4. **操作**：预览、下载、删除（链接按钮样式）

**多文档在同一行横向换行**：

- 外层 `.list.list`：`flex-direction: row` + `flex-wrap: wrap` + 行/列 `gap`。
- 每条 `.doc-card--list`：`inline-flex`、`width: max-content`、`max-width: 100%`，使多条并排；窄屏下单块不超过容器宽度。

**备注**：

- 若文档带 `remark`，在**主行下方**单独一行展示（`.doc-remark-line`），整块仍视为一个换行单元。

### 3.4 空态与加载

- 加载中：居中提示文案。
- 无数据：居中「暂无文档」类提示。

## 4. 上传逻辑与可配置项（与实现一致）

### 4.1 Props（DocumentUploadPanel）

| 属性 | 说明 | 默认值（组件内） |
|------|------|------------------|
| `bizType` | 业务类型，随上传接口传递 | 必填 |
| `bizId` | 业务主键 | 必填 |
| `maxFiles` | 单次最多文件数 | `5`（业务页可覆盖，如采购单 `20`） |
| `maxSizeMb` | 单文件最大 MB | `50`（业务页可覆盖，如 `100`） |
| `remarkAllowed` | 是否显示备注输入 | `true` |

### 4.2 允许扩展名

`.pdf`、`.jpg`、`.jpeg`、`.png`、`.docx`、`.xlsx`、`.zip`

不符合扩展名或超过大小、超过数量时：**提示**（如 `ElMessage.warning`），跳过非法项；数量达上限时停止追加。

### 4.3 事件与对外方法

- 上传成功：`emit('uploaded', documents)`，由业务页刷新关联列表（如调用 `DocumentListPanel` 的 `refresh`）。
- `defineExpose({ addDroppedFiles })`：供外层容器（如整 Tab）在 `drop` 时将 `File[]` 转入与组件内部相同的校验与队列逻辑。

## 5. 整 Tab 拖放（业务页集成约定）

当详情页「文档」Tab 内同时包含**上传面板**与**关联文档列表**时，建议：

- 在外层容器（如 `.doc-tab-content`）上监听 `dragover` / `dragleave` / `drop`，与上传组件类似做深度计数高亮（如 `doc-tab-content--dragging`）。
- `drop` 时取 `dataTransfer.files`，调用上传子组件的 **`addDroppedFiles`**，使拖到列表空白处、备注区附近等区域仍进入同一套待上传队列。

注意：上传组件根节点上的拖放事件应 **`stop` 冒泡**，避免与外层重复处理导致文件被添加两次。

## 6. 自检清单（实现 / 走查）

- [ ] 未选中文件时不显示备注输入；选中文件后备注显示在待选文件列表下方
- [ ] 备注无外侧标题，placeholder 为「请输入备注，选填，最多256字」
- [ ] 拖拽区无图标；上传组件外层浅灰边框（`#dcdfe6`）；提示文案单行并排
- [ ] 工具行：拖拽区 → 上传/清空，**垂直居中**
- [ ] 上传区高度采用紧凑规格（主行/上传区最小高度 `52px`）
- [ ] 拖拽提示文案字号为 `13px`（与页签标题一致）
- [ ] 待选文件：多文件横向换行，组间距清晰
- [ ] 关联文档：`刷新` 在「关联文档」标题右侧
- [ ] `list` 模式：每条主行「文件名 | 日期 | 字节(千分位) | 操作」，多条卡片**同一行横向换行**
- [ ] `grid` 模式：仍为缩略图卡片，未强制改为列表条
- [ ] 整 Tab 拖放（若启用）：高亮 + 仅一次入队
