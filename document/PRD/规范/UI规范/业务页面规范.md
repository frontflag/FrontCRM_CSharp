# 业务页面规范（总览）

本文档对**业务相关前端页面**（以列表、详情、筛选与表格行为主）所遵循的 UI 规范做**索引与引用**，具体条款以各子文档为准。

**适用范围：** 客户、供应商、询价/订单、库存、财务等业务模块中的列表页、详情页及其共用组件；实现时与代码注释中提及的《业务列表规范》等表述一致时，以对应链接文档为准。
