# UI规范总索引

本目录用于沉淀前端 UI 统一规范，作为列表页与表单页的实现基线。

## 已发布规范

- [列表字段宽度规范PRD](./列表字段宽度规范PRD.md)
  - 目标：统一列表字段 `width / min-width / tooltip` 的配置方式，保障可读性与一致性。

- [列表操作按钮颜色规范PRD](./列表操作按钮颜色规范PRD.md)
  - 目标：统一操作列按钮颜色语义（查看/编辑=蓝，高危=红，业务操作=非蓝非红）。

## 使用建议

1. 新建列表页前，先对照本目录规范。
2. 代码评审时，将本目录作为 UI 一致性验收依据。
3. 新增规范后，务必同步更新本索引。
