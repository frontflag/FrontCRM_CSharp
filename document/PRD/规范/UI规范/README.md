# UI规范总索引

本目录用于沉淀前端 UI 统一规范，作为列表页与表单页的实现基线。

## 已发布规范

- [列表字段宽度规范PRD](./列表字段宽度规范PRD.md)
  - 目标：统一列表字段 `width / min-width / tooltip` 的配置方式，保障可读性与一致性。

- [列表操作按钮颜色规范PRD](./列表操作按钮颜色规范PRD.md)
  - 目标：统一操作列按钮颜色语义（查看/编辑=蓝，高危=红，新建/新增/创建=绿，业务流转=非蓝非红非绿，常用黄/灰）。

- [列表操作列视觉规范PRD](./列表操作列视觉规范PRD.md)
  - 目标：统一操作列区域底色（`$layer-3`）、左侧 inset 分隔线、行 hover 叠青、`.action-btns` 默认隐藏与行悬停显隐；并约定 `el-table` 固定右列与原生表末列对齐方式。

- [客户供应商名称状态展示规范PRD](./客户供应商名称状态展示规范PRD.md)
  - 目标：冻结或黑名单时名称灰色；冻结时在名称后加雪花图标，黑名单时加骷髅图标；统一列表/详情/黑名单展示与组件约定。

- [列表复选篮子批量操作规范PRD](./列表复选篮子批量操作规范PRD.md)
  - 目标：凡「分页 + 多选 + 批量操作」的列表，统一使用「复选篮子」（跨页保留、页脚左对齐入口、抽屉查看/移除、清空确认、批量动作以篮子为准）；参考 `RFQItemList.vue` 与 `rfqItemListBasket.ts`。

- [扩展面板.帮助规范](./扩展面板.帮助规范.md)
  - 目标：约定右侧「帮助」扩展面板（扩展面板.帮助）的 Markdown 编写结构（页面功能 / 操作说明卡片块）、状态用语义名称、省略通用操作说明；以及与 `HelpManualPanel.vue` 中 `help-md` 标题层级与 `.help-op-block` 等展示样式一致；参考 `销售订单_MENU_SALES_ORDERS.md`。

## 使用建议

1. 新建列表页前，先对照本目录规范。
2. 代码评审时，将本目录作为 UI 一致性验收依据。
3. 新增规范后，务必同步更新本索引。
