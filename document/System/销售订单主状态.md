# 销售订单主状态（`sellorder.status`）

C# 枚举：`CRM.Core.Models.Sales.SellOrderMainStatus`（`: short`，与数据库 `smallint` 一致）。

| 数值 | 枚举成员 | 含义 |
|------|----------|------|
| 1 | `New` | 新建 |
| 2 | `PendingAudit` | 待审核 |
| 10 | `Approved` | 审核通过 |
| 20 | `InProgress` | 进行中 |
| 100 | `Completed` | 完成 |
| -1 | `AuditFailed` | 审核失败 |
| -2 | `Cancelled` | 取消 |

- 与 `SellOrder`、`SalesOrderService.UpdateStatusAsync` 校验一致：`New(1)→PendingAudit(2)→Approved(10)/AuditFailed(-1)` 等。
- 待办审批（`ApprovalsController` / `SALES_ORDER`）：`PendingStatus = **2**`（`PendingAudit`），通过为 **10**（`Approved`），驳回为 **-1**（`AuditFailed`）。
- 申请出库：`RequestStockOutAsync` 将主状态置为 **InProgress（20）**；出库业务校验「已完成」为 **Completed（100）**。
- 若库中仍为历史编码，按迁移脚本（如 `migrate_sellorder_status_to_10_20_100.sql`）统一到上表。
