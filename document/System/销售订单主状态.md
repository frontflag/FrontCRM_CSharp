# 销售订单主状态（`sellorder.status`）

C# 枚举：`CRM.Core.Models.Sales.SellOrderMainStatus`（`: short`，与数据库 `smallint` 一致）。

| 数值 | 含义 |
|------|------|
| 0 | 新建 |
| 1 | 待审核 |
| 10 | 审核通过 |
| 20 | 进行中 |
| 100 | 完成 |
| -1 | 审核失败 |
| -2 | 取消 |

- 与 `SellOrder`、`SalesOrderService.UpdateStatusAsync` 校验一致。
- 待办审批（`ApprovalsController` / `SALES_ORDER`）：待审=**1**，通过=**10**，驳回=**-1**。
- 申请出库：`RequestStockOutAsync` 将主状态置为 **InProgress（20）**；出库业务校验「已完成」为 **Completed（100）**。
- 若库中仍为旧编码（如 2/3/4 表示 审核通过/进行中/完成），执行 `migrate_sellorder_status_to_10_20_100.sql`。
