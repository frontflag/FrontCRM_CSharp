# 客户列表

页面路径：`/customerlist`（旧路径 `/customers` 会重定向至此）。

## 功能说明

在此页面可查看、检索客户，并进入详情或编辑。

## 检索与筛选

- **关键词**：支持按客户名称、联系人等检索（视权限可能仅限编号）。
- **类型 / 行业**：下拉筛选后点击「搜索」或变更即触发查询。
- **仅收藏**：只显示已收藏的客户。

## 左侧扩展栏

- **检索**：与列表相同条件，纵向表单；搜索后跳转本页并带查询参数。
- **收藏 / 历史**：快速打开常用客户。

## 操作

- **详情 / 编辑**：在表格行或操作列进入对应页面。
- **提交审核**：新建状态客户可提交进入审批流程（需权限）。
